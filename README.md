#	artDialog``v6``
==========================

artDialog v6 —— 经典的网页对话框组件，内外皆用心雕琢。

##	用户

四年来，有超过 40 万网站在使用 artDialog，其中包括：

*	[QQ空间 v8（腾讯）](http://qzone.qq.com)
*	[QQ旋风（腾讯）](http://xf.qq.com)
*	[Phpcms（盛大）](http://www.phpcms.cn)
*	[极路由](http://www.hiwifi.com)
*	[中国电信客服系统](http://www.189.cn)

##	源码说明

*	./popup.js	``基础浮层类``
*	./dialog/dialog.js	``对话框主脚本``
*	./dialog/dialog-iframe.js	``简化创建 iframe 内容对话框的版本``
*	./dialog/dialog-drag.js	``支持拖拽的插件，引入即可支持拖拽``
*	./dialog/doc/	``对话框文档目录``
*	./dialog/test/	``对话框测试目录``
*	./dialog/css/	``对话框样式目录``
*	./dialog/drag/drag.js	``基础拖拽类``
*	./lib/	``依赖的外部库目录(jQuery、RequireJS、SeaJS)``
*	./selectbox/selectbox.js	``下拉控件装饰器``
