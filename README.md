#	PopupJS

基于 [W3C HTML5 Dialog API](http://www.w3.org/TR/2013/CR-html5-20130806/interactive-elements.html#the-dialog-element) 的浮层基础类。

适用于：对话框、下拉菜单、Tips 等 UI 组件的基础类。

##	特性

1. 基于 HTML5 Dialog API
2. 提供 12 中方式的元素对齐或页面居中显示
3. 自动 zIndex
4. 完善的焦点处理，自动焦点附加与回退
5. 支持普通与模态浮层
6. 支持 ARIA 标准
7. 兼容 IE6 与所有主流浏览器
